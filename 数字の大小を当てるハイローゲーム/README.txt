プログラム説明：
このプログラムは、ユーザーがCOMの出す数字が大きいか小さいかを予想する「ハイローゲーム」です。
ユーザーは0か1を選択することができます。0を選択すると、「Small」、1を選択すると「Big」として扱われます。
COMも数字を出し、互いに出した数字の大きさを比較します。
同じ場合は引き分け、予想が当たれば勝ち、外れれば負けとなります。

プログラムは、3回までプレイできます。プレイ回数が3回に達すると、ゲームは自動的に終了します。
また、ユーザーが「もう一度プレイしますか？」の質問に「Yes」を選択した場合、ゲームを続けることができます。


Program Outline:
This program is a simple "High & Low" game where the user tries to guess whether 
a random number generated by the computer is bigger or smaller than the number they themselves generate.

The program uses a while loop to allow the user to play the game multiple times if they wish. 
It generates two random numbers, one for the computer and one for the user, 
and compares them to determine the winner. 
If the user wins, "Win!" is printed to the console. If the computer wins, "Lose.." is printed.
If the numbers are the same, "Draw" is printed.

The program also keeps track of the number of times the game has been played, 
and if the user has played three times, it will end the game.
